Digital Logic SIM es una herramienta de simulación de circuitos lógicos digitales que permite a los usuarios diseñar, analizar y probar circuitos digitales en un entorno virtual. Guía Completa para Crear un Repositorio Local en Git y Conectarlo a GitHub

Git es una herramienta esencial para el control de versiones, permitiendo a los desarrolladores gestionar y colaborar en proyectos de manera eficiente. En esta entrada, aprenderemos cómo crear un repositorio local, trabajar con ramas y conectarlo a un repositorio remoto en GitHub.

Crear un Repositorio Local

Para crear un nuevo repositorio local, sigue estos pasos:

  1. Instalar Git: Si aún no tienes Git instalado, descárgalo e instálalo desde git-scm.com.
  2. Inicializar el Repositorio: Abre tu terminal o línea de comandos, navega hasta el directorio donde deseas crear tu proyecto, y usa: 
    git init nombre-del-repositorio
  3. Añadir Archivos: Agrega archivos a tu nuevo repositorio: 
    touch archivo1.txt archivo2.txt
    Luego, añade los archivos al índice de Git: 
    git add .
  4. Realizar un Commit: Ahora que has añadido los archivos, realiza tu primer commit: 
    git commit -m "Primer commit"

Crear Ramas (Branches) en Git

Las ramas son fundamentales en Git. Para crear una nueva rama:

git branch nombre-de-la-rama

Cambiar a la nueva rama:

git checkout nombre-de-la-rama

Ejemplo práctico: Si trabajas en una característica llamada "autenticación", puedes crear una rama así:

git checkout -b autenticacion

Combinar Ramas (Merge)

Una vez que hayas terminado de trabajar en tu rama, cambia a la rama principal y combina:

git checkout main
git merge autenticacion

Conectar tu Repositorio Local a GitHub

Para conectar tu repositorio local a GitHub:

  1. Crea un repositorio en GitHub y copia la URL.
  2. En tu terminal, añade el repositorio remoto: 
    git remote add origin https://github.com/tu-usuario/nombre-del-repositorio.git
  3. Sube tus cambios: 
    git push -u origin main
